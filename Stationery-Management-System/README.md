Stationery Management System
Description
A console-based C++ application for managing book records in a library or stationery system.
Features
•	Add new books
•	View book information
•	Search books by author
•	Display book titles
•	Count total books
Technologies
•	C++
•	Structures
•	Console Application
Concepts Used
•	Structures
•	Arrays
•	Searching
•	Functions
•	Loops
•	Conditional Statements
How to Run
1.	Compile the source file.
2.	Run the executable.
3.	Select an option from the menu.
4.	Manage book records.
