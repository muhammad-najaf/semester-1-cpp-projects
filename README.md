Semester 1 C++ Projects
This repository contains the C++ console-based projects I developed during my 1st Semester of the Bachelor of Software Engineering program.
These projects were created to strengthen my understanding of fundamental programming concepts, problem-solving, and object-oriented thinking using C++.
Projects Included
1. Car Rental System
A console-based application that simulates a basic car rental service.
Features
•	User registration and login
•	Car selection
•	Credit card balance validation
•	Rental payment processing
•	Invoice generation
•	Random Order ID generation
•	Multiple rentals in one session
Concepts Used
•	Classes and Objects
•	Functions
•	Loops
•	Conditional Statements
•	Random Number Generation
•	User Input Validation

2. Examination Management System
A console application for managing student examination eligibility.
Features
•	Add student records
•	Calculate attendance percentage
•	Check examination eligibility
•	Delete student records
•	Update eligibility criteria
•	Print all student records
Concepts Used
•	Structures
•	Arrays
•	Functions
•	File-like Record Management
•	Loops
•	Conditional Statements

3. Stationery Management System

Features

Concepts Used
