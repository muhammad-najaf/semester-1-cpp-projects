Car Rental System
Description
A console-based C++ application that simulates a basic car rental service. Users can register, log in, choose a car, complete payment, and receive an invoice.
Features
•	User registration
•	User login authentication
•	Car selection
•	Credit card balance verification
•	Payment processing
•	Invoice generation
•	Random order ID generation
•	Multiple rentals in one session
Technologies
•	C++
•	Object-Oriented Programming
•	Console Application
Concepts Used
•	Classes and Objects
•	Functions
•	Loops
•	Conditional Statements
•	Random Number Generation
•	User Input Handling
How to Run
1.	Compile the source file.
2.	Run the executable.
3.	Register a user account.
4.	Log in using the registered credentials.
5.	Select a car and complete the rental process.
