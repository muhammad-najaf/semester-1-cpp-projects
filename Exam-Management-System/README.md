Examination Management System
Description
A console-based C++ application for managing student examination records and determining examination eligibility based on attendance and fee status.
Features
•	Add student records
•	Calculate attendance percentage
•	Check examination eligibility
•	Delete student records
•	Update eligibility criteria
•	Display all student records
Technologies
•	C++
•	Structures
•	Console Application
Concepts Used
•	Structures
•	Arrays
•	Functions
•	Loops
•	Conditional Statements
How to Run
1.	Compile the source file.
2.	Run the executable.
3.	Add student records.
4.	Manage and view examination eligibility through the menu.
